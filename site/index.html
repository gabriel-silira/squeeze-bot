Oque faz os comandos

comando dev: Mostra quem fez o bot

comando squeeze: O bot envia uma saudação

comando ping: Mostra o ping do bot

comando data: Mostra a data atual

comando hora: Mostra a hora atual

comando moeda: Gira um Cara ou Coroa

comando d6: Rola um dado de 1 a 6 números

comando d20: Rola um dado de 1 a 20 números

comando d100: Rola um dado de 1 a 100 números

comando piadas: Envia uma piada aleatória entre 500
