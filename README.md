bot_oliveira

projeto de bot de entretenimento

link do site: https://gabriel-silira.github.io/oliveira-bot/site/index.html

comandos

dev: mostra o criador
oliveira: uma saudação
ping: devolve o ping do bot
data: a data atual
hora: a hora atual
moeda: tira cara ou coroa
d6: rola um dado de 6 lados
d20: rola um dado de 20 lados
d100: rola um dado de 100 lados
piada: carrascosa
piadas: manda uma piada aleatória de uma lista com 501
